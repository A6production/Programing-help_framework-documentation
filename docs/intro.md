# Documentation Introduction
This is the official Programing-help Framework documentation. This is an open source python library for developers with a lot of functions.

If you have any questions you can ask in Github Discussions

If you encounter / found any issues you can tell us about them in Github Issues

This is an open source library, so you can always fork it or make a pull request that can later be merged and applied in the new version

## Installation
```shell
pip install https://github.com/ProgramingHelp/Programing-help_Framework 
```

## Upgrade
```shell
pip install https://github.com/ProgramingHelp/Programing-help_Framework --upgrade
```
